THE CARD CATALOGIndex — pointers, not content

The Card Catalog — gbrain index

boot / gbrain sync reads the vault incrementally and builds gbrain — a local PGLite index (keyword + vector embeddings via a ZeroEntropy key). It does not hold your content — it points at the files.

Side belt: the capture line can also write straight into gbrain's searchable store via gbrain put.

The Pickers — AI Agent

An AI agent (Claude Code / Hermes) reaches gbrain through a read-only MCP search tool and answers your questions in your own cited words — pulling the source files back off the shelf.

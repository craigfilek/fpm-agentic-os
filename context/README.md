# context/

What the system knows about **you** and your work. Filled by `/onboard`; you can
edit any file here directly. Plain markdown — the source of truth.

Typical files (created by `/onboard`):
- `about-you.md` — who you are, what you do, how you spend your time.
- `voice.md` — real samples of how you write, so every agent sounds like you.
- `vision.md` — what your work is for; your priorities.

If gbrain is installed, these are also indexed so the agent can recall over them.

# Welcome to your second brain

This is your first note. It lives in `Inbox/` — the drop zone for anything new.

**Two folders, two jobs:**
- `Inbox/` — drop raw notes, links, transcripts here. Capture first, sort later.
- `Distilled/` — refined, long-lived notes worth keeping. Your canon.

**The loop:**
1. Drop a file in `Inbox/` (or run `capture <file-or-url>`).
2. Run `boot` — it indexes new notes and shows a green health lamp.
3. Ask your agent to recall by meaning: *"what did I save about X?"*

Delete this note whenever you like. The brain keeps working.
